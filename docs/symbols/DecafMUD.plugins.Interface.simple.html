DecafMUD.plugins.Interface.simple(decaf)

This is a minimal user interface for DecafMUD, only providing a basic input handler if an input element is provided and rendering output to a display.

Generally, you'll want to use the full interface for a richer user experience.

Parameters:
{DecafMUD} decaf
The instance of DecafMUD using this plugin.

infoBar(text, clss, timeout, icon, buttons, click, close)
Create a new notification bar at the top of the interface for the user to take action on. Actions may be specified to be taken when the bar is clicked or closed, and buttons may be added as well. If the second parameter is a number instead of a string, it will be treated as though timeout and clss have swapped places.
Parameters:
{String} text
The text to display on the bar.
{String} clss Optional, Default: "info"
Optionally, a class to add to the bar for more precise styling.
{Number} timeout Optional, Default: 0
The number of seconds after which the bar should automatically be closed.
{String} icon Optional
The URL of an image to display on the bar.
{Array} buttons Optional
A list of buttons to be displayed.
{function} click Optional
A function to be called when the bar is clicked.
{function} close Optional
A function to be called when the bar is closed.
