Class Detail
DecafMUD.plugins.Display.standard()
This is the standard display handler for DecafMUD, and should generally be good enough for just about anything you'd need to do. It has support for many ANSI sequences, XTERM-style 256 colors, MXP, multiple output panes, and limiting available scrollback for performance reasons. 
var ESC = "\x1B";
decaf.display.handleData(ESC + "[0m"); // Reset the ANSI SGR Settings
