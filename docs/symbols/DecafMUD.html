Convert a telnet IAC sequence from raw bytes to a human readable format that can be output for debugging purposes. 
var IAC = "\xFF", DO = "\xFD", TTYPE = "\x18";
DecafMUD.debugIAC(IAC + DO + TTYPE);

The sequence to convert.
Returns:
{String} The human readable description of the IAC sequence.

requestPermission(option, prompt, callback)
Request permission for a given option, as stored in the global settings object at the given path. This will ask the user if they want to allow an action or not, provided they haven't given an answer in the past. Since the user input may take some time, this will call the provided callback function with the result when the user makes a decision.
Parameters:
{String} option
The path to the option to check.
{String} prompt
The question to show to the user, asking them if it's alright to do whatever it is you're doing.
{function} callback
The function to call when we have an answer.
require(module, check)
Require a moddule to be loaded. Plugins can call this function to ensure that their dependencies are loaded. Be sure to use this along with waitLoad to ensure the modules are loaded before calling code that uses them. 
decaf.require('decafmud.encoding.cp437');
// External module
decaf.require('my-module', function() {
   return 'SomeRequirement' in window;
});
Parameters:
{String} module
The module that has to be loaded.
{function} check Optional
If specified, this function will be used to check that the module is loaded. Otherwise, it will be looked for in the DecafMUD plugin tree.
